使用Typora写Markdown

常用操作

[toc]

标题：1-6级

  • 快捷方式：Ctrl+1-6
  • 转义字符：#的数量表示级数 注意#和文本有个空格
  • 四个#号控制的标题-四级标题 #### 四级标题

序列：

  • 快捷方式

    1. 有序列表 Ctrl+shift+[
    2. 无序列表 Ctrl+shift+]

  • 转义字符
    有序列表 在空白的一行输入 1.空格 即可实现缩进排序的效果
    无序列表 在空白的一行输入 -空格 空格可以理解为转义字符

  • 先enter进入下一个序列；一行未输入内容前 按下tab键可以进行二级列表， 即列表的列表 （以此类推）

    • 二级列表

字体格式

Ctrl+B/I/U = 加粗/斜体/下划线

删除线

  • 删除线~~~ 双波浪可以和高亮显示的双等号一起记忆 ~~无快捷键
  • $公式$ $e^2$

任务列表

  • 语法：空格-空格[ 内1空格 ]空格

注释

只会在预览时显示 导出图片 /pdf等文件不会显示

Emoji图标

  • 空格:代号 :后面跟Emoji图标的代号 代号不是很严格 有联想功能 例如 :up 显示如下
  • 是一个小手朝上的emoji

分割线

— 在空白一行输入三个（或以上）-或者*即实现分割线的效果

进阶操作

表格

  • 快捷键 Ctrl+T
  • 转义字符特别复杂 不如用快捷键

特殊块

  • Ctrl+Shift+K = 代码

  • Ctrl+Shift+M = 公式

  • Ctrl+Shift+Q = 引用

代码模式

  • 代码模式切换 Ctrl+/
  • 顺口溜：尖波浪双等号
  • ^ ^ 我是^上标^
  • ~ ~ 我是下标
  • == == ==高亮==

插图

  • 快捷键 Ctrl+shift+i
  • 立马转化为在小括号中选取文件路径

超链接

  • 快捷键 Ctrl+K
  • 链接提示语 同插图理 小括号里面放入目的网址的路径
  • 我的博客主页

表格：代码模式

代码模式 （这个必须切换到代码模式才能看清楚到底发生了什么 第二行不可少 虽然是空的）

| 姓名 | 性别 | 生日 |

| —- | —-| ——– |

| 张三 | 男 | 19970101 |

| 李四 | 男 | 19970601 |

效果如下表所示：

姓名 性别 生日
张三 男 19970101
李四 男 19970601

代码块

  • ```+代码类别
  • ```三飘号(数字1左边的那个下符号)+代码标题+enter+enter 切换到代码模式
  • 第一个enter是确认标题 第二个enter是激活代码模式 第二个才是核心

引用

>+内容 在>的后面输入内容即可 但是>需要在一行的最前端

人望山，鱼窥荷。

参考文献

[^数字] 正文中进行引用的语法（内容写在前） 解释：^后面跟数字自动转换成引用格式[]

[^数字]： 与之对应的文末文献简介的语法 解释：原有基础上加个冒号：即可

目录生成

[toc] 后面跟回车即可变换为显示的目录样式 toc是table of contents的缩写

流程图

语法注释：```+flow 关键字来实现流程图样式的转义 还有其他关键字来显示其他形式流程图

底层操作

Html+CSS

从底层代码的角度修改CSS代码文件来实现高度自定义的主题样式

  1. 从官网下载目标样式的模板文件夹
  2. 把文件夹里面的CSS文件粘贴出来与文件夹并列
  3. 打开CSS文件 通过修改代码 局部微调主题样式、大小 、字体、 颜色之类

一些快捷键

新建 CTRL + N

保存 CTRL + S

另存为 CTRL + SHIFT + S

一级标题: CTRL + 1

二级标题: CTRL + 2

三级标题: CTRL + 3

四级标题 : CTRL + 4

五级标题 : CTRL + 5

六级标题 : CTRL +6

插入表格: CTRL + T

插入引用: CTRL + SHIFT + Q

有序列表: CTRL + SHIFT + [

无序列表: CTRL + SHIFT + ]

代码块: CTRL + SHIFT + K

字体加粗: CTRL + B

超链接: CTRL + K

插入图片: CTRL + SHIFT +l

显示/隐藏侧边栏: CTRL + SHIFT + L

打开/关闭源代码模式: CTRL + /

全文查找: CTRL + F

全文替换: CTRL + H

开发者工具: SHIFT + F12

跳转到文首: CTRL + Home

跳转到文末 : CTRL + End

注：上面跳转文首、文末 HOME键 和 End键位置：（右边上下左右键的 上方有6个键，中间2个就是Home和End键）
